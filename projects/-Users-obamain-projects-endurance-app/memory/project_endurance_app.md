Building an endurance athlete OS for committed amateurs (5+ hrs/week, racing occasionally).

**Wedge (build first):** Rehab for Runners — biggest audience, fastest validation signal, near-free SEO distribution.

**Platform anchor (long-term):** Nutrition for Triathletes — higher LTV, natural upgrade path from running → triathlon.

**Metronome/BPM:** Feature inside the running product, not a standalone product.

**Why:** Tri Plans + Content was dropped (too competitive). Runner rehab is the fastest path to first signal and revenue.

**How to apply:** Frame all suggestions — features, channels, priorities — through the runner rehab wedge first. Don't suggest triathlon features until the running product has validated demand.
